This is an implementation of a Priority Queue in Java by Neelay Junnarkar

This priority queue uses a binary heap represented as a 1-dimensional array to allow for efficient algorithms.

Complexities of Algorithms
======
    boolean offer(T): O(log n)
    boolean add(T): O(log n)

    T poll(): O(log n)
    T remove(): O(log n)

    boolean remove(T): O(n)
    boolean contains(T): O(n)

    T peek(): O(1)
    T element(): O(1)

    int size(): O(1)
